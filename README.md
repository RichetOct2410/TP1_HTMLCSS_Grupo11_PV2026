El proyecto es un Gestor Academico que cuenta con varias funciones como una seccion de estadisticas, otra de novedades.
Un explorador de proyectos que tiene un listado de proyectos con una barra de filtros, util para encontrar el proyecto buscado por el usuario.
Cada proyecto cuenta con un encabezado, una descripcion extendida, la lista de recursos y una seccion de equipo que describe el rol que cumple cada integrante.
Por ultimo el perfil de usuario muestra informacion personal del usuario y su rol.
